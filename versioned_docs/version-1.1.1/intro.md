# Medusa Documentation

Welcome to the Medusa documentation site.

This repository is designed to be updated automatically from code pull requests using gh-aw workflows.

## How this repo is used

- Developer pull requests in the code repository trigger an automated docs analysis.
- gh-aw proposes docs updates here as pull requests.
- Release workflows update release-note content and support versioned snapshots.

## Authoring guidelines

- Keep sections practical and user-focused.
- Link to relevant pull requests or issues when possible.
- Add migration notes whenever behavior changes.
